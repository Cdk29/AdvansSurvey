# AdvansSurvey

## Requirements
+ Python 3.4, 3.5, 3.6 or 3.7
+ Pip package installe

## Django installation

> python -m pip install Django

## Django launching

You have to be in the same file as the manage.py script to type this command.
> python manage.py runserver
